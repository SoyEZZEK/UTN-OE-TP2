# TP2 - Gestión Colaborativa, Control de Versiones y Organización Empresarial

## Descripción
Trabajo Práctico 2 de Organización Empresarial - UTN TUPaD.
Implementación de un flujo de trabajo colaborativo utilizando Git, GitHub y Jira,
aplicado al análisis de datos de ventas de una pequeña empresa (escenario B).

## Integrantes
- Lautaro Pérez (Roles: Hugo - Paco - Luis)

## Escenario elegido
Escenario B - Análisis de Ventas de una Pequeña Empresa

## Dataset utilizado
Dataset de ventas simuladas con registros de fecha, producto, cantidad y precio.
Fuente: https://gist.github.com/khanusama20/ee33c2869dd5cf3cebdf020be1ca43f6

## Estructura del proyecto
- /datos -> Dataset utilizado en el análisis
- /scripts -> Script Python de análisis de ventas
- /resultados -> Gráficos y resultados generados

## Instrucciones para ejecutar
1. Abrir Google Colab
2. Clonar el repositorio
3. Ejecutar el script: scripts/analisis_ventas.py
